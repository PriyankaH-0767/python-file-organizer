# Smart File Organizer

## Project Description

This Python application automatically organizes files into categories such as Images, Documents, Videos, Audio, and Others.

## Features

* Organizes files automatically
* Creates folders if they do not exist
* Maintains a log file
* Handles errors using exception handling
* Accepts user input for folder path

## Technologies Used

* Python
* OS Module
* Shutil Module

## How to Run

1. Open terminal
2. Run:
   python organizer.py
3. Enter the folder path
4. Files will be organized automatically

## Output

Files are moved into categorized folders and actions are recorded in log.txt.
